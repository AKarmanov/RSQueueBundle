RSQueue

Un sistema de colas sencillo, rápido y ágil, basado en redis
para desarroladores de Symfony2

El proyecto se ha creado completamente open source. Eres completamente libre de modificar el código y compartirlo con la comunidad.

El código está documentado siguiendo el PHPCS de symfony2 de forma completamente estricta. Nos gusta el código limpio y estructurado.

El proyecto está basado en el sistema clave-valor redis. Utiliza el bundle de redis snc/redis-bundle.

Installation

Compatible with Symfony 2.3 and later

Install redis

First of all, you must install redis service in your system. Otherwise, this bundle will not work. You can find a very intuitive and elaborated guide here.

wget http://download.redis.io/redis-stable.tar.gz
tar xvzf redis-stable.tar.gz
cd redis-stable
make

Install RSQueueBundle

Once you have redis installed properly, you need to add a require line with RSQueueBundle

"require": {
"php": ">=5.3.3",
"symfony/symfony": "2.3.*",
...
"mmoreram/phpcs-bundle": "dev-master"
},

Then you need to use composer to update your project dependences

php composer.phar update

And register the bundle in your appKernel

return array(
// ...
new Mmoreram\RSQueueBundle\RSQueueBundle(),
// ...
);
